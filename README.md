# ArteryMusic

#### 介绍
一个简单的在线音乐系统（动脉音乐网站）

#### 软件架构
Java

Javaweb

Servlet

Jsp

mybatis

mysql

***需要引入的jar包***

```java
commons-dbutils-1.7.jar
javax.servlet-api-4.0.1.jar
jstl-1.2.jar
mybatis-3.5.11.jar
mysql-connection-j-8.0.32.jar
```


#### 安装教程

1.  执行SQL，创建数据库
2.  项目导入IDEA 启动
3.  访问测试！
4.  自行根据代码逻辑填充测试数据，查看即可！
